You can fully customize your WordPress login page with Easy Custom Login plugin.


== Description ==

Tired of watching the same WordPress login page every time? Need to change logo at the top of login form? Want custom background on login page? Want lots of other really necessary stuffs? This plugin is the best and most easy way to customize your WordPress default login page.

**Features**<br>
1. Set custom background image<br>
2. Upload or give custom image URL for custom login background<br>
3. Set custom color as login page background<br>
4. Set custom logo above login form<br>
5. Upload or give custom image URL for custom login logo<br>
6. Remove "Remember Me" link<br>
7. Remove "Lost Password" link<br>
8. Remove "Back To..." link<br>
9. Set custom URL where users will be redirected after LOGIN<br>
10. Set custom URL where users will be redirected after LOGOUT<br>
11. Add custom message over the login box<br>
12. Set custom font color for login box text<br>
13. Set login box background color


== Installation ==

**Way 1:**<br>
1. Download the plugin file<br>
2. Unzip the file into a folder on your hard drive<br>
3. Upload via FTP into your plugins (/wp-content/plugins/) folder<br>
4. Activate the plugin from your dashboard<br>

**Way 2:**<br>
1. From Dashboard > Plugins > Add New > Upload the plugin<br>
2. Activate the plugin<br>
